Aplicações para internet

A matéria consiste em explicar como funciona a internet e as tecnologias utilizadas para criar websites.

História da Web e Internet

Antes da Web atual existia o Bulletin Board System (BBS) que se tratava de uma ferramenta corporativa que permitia
usuários locais se comunicarem, toda a comunicação era feita através de uma linha telefônica.
De princípio, quando criada em 1978 por Walt Christensen, em Chicado, era apenas para trocar mensagens, mas com o
tempo se aperfeiçou e foi possível trocar arquivos e abrir foruns.

Ela funcionava através de uma criação de um BBS por um operador de sistema, também chamado de SysOp, em sua pró-
pria máquina, e uma vez criada ele disponibilizava para a comunidade local. A BBS começou a perder popularidade em
1990 com a criação da World Wide Web (WWW).

Mas a internet de fato iniciou nas decadas de 60 quando a ARPANet foi criada com o intuito de disponilizar comunica-
ção entre os centros de pesquisas e o Departamento de defesa do governo dos EUA.
Com a expansão da ARPANet foi necessário estabelecer uma linguagem comum entre as diferentes redes, e nesse con-
texto foi criado em 1973 o protocolo TCP/IP por Vint Cerf e Robert Kahn.

Em 1983 o DNS (Domain Name Server) criado por Paul Mockapetris surge e, em 1984 o TCP/IP torna-se o padrão da
internet pela CERN.

Com o tempo a internet criada pela ARPANet começou a se expandir mais e mais e foi criado um movimento em 1988
para tornar a internet comercial, permitindo a possibilidade de utiliza-la fora dos centros de pesquisas. Foram surgindo
os primeiros ISP (Internet Service Provider), aquelas empresas que fornecem o acesso a internet.
Empresas que fornecem a internet através da linha telefônica (dial-up) ou banda larga. Mas somente em 1993 a internet
ser tornou de uso comercial de fato.

A World Wide Web teve seu início em 1989 e foi criada na Europa pelo cientista Tim Berners-Lee. O primeiro site, na-
vegador e também o primeiro servidor WEB foram criados em 1990, o servido foi chamado por Tim de "NeXT".

O primeiro servidor fora da Europa foi criado em 1991 na Califórnia e era chamado de SLAC e em 1992 foi criado o
primeiro navegador multiplataforma (em modo linha de comando). Mosaic 1.0 foi lançado em 1993 por NCSA da Uni-
versidade de Illinois.

Mas a WEB só foi ganhar popularização mesmo em 1994, após a CERN declarar a WEB como domínio público em 1993.

Funcionamento da internet

O funcionamento da internet consiste na busca de alguma informação de um usuário comum em algum servidor. Um usuá-
rio comum, solicita a informação e o servidor retorna essas informações, que normalmente são enviadas através da Uniform
Resource Locator (URL), que seria o endereço para um recurso da internet. Na URL contém o identificador do protocolo, o
nome do recurso que contém o nome do host, do arquivo e também a porta de serviço e a referência.

Por Exemplo:

protocolo://domínio:porta/caminho/recurso?query_string#referência
https://www.cruzeirodosul.edu.br/pesquisa.jsp?q=java

O que é HTML?

HTML Hypertext Markup Language ou Linguagem de Marcaçao de Hipertexto é um documento de texto que pode ser produzi-
do utilizando qualquer editor de texto e é utilizado por todas as páginas WEB disponíveis na internet.
O HTML define um conjunto de elementos para a marcação de uma página e cada elemento possui uma função diferente, elemen-
tos que são normalmente chamados de tag (marcadores).
Foi desenvolvida por Tim Berners-Lee o criador da World Wide Web (WWW) em 1991.

O HTML passou por diversas modificações até chegar na sua versão atual, breve histórico:

* 1991: HTML 1 (Tim Berners-Lee);
* 1994: World Wide Web Consortium (W3C) a padronizadora das tecnologias WEB;
* 1995: HTML 2;
* 1997: HTML 3.2 e HTML 4;
* 1998: HTML 4 se torna oficial;
* 1999: HTML 4.01
* 2000: XHTML 1.1 se torna padrão junto com o HTML 4.01;
* 2008: HTML 5;
* 2017: HTML 5.2 e 5.3 com a recomendação da W3C.

O HTML assim como algumas outras tecnologias, tem grandes avanços a partir de sua comunidade, com o HTML não é diferente, ele
teve particiapação de empresas e pessoas que se interessaram pelo projeto e quiseram seu avanço.
Assim como o Web Hypertext Application Technology Working Group (WHATWG), que é um grupo de pessoas e empresas interessa-
das no desenvolvimento do HTML e APIs, esse grupo foi fundado por representantes da Apple, Mozilla e Opera em 2004, depois de re-
alizarem um workshop com a W3C.

TAG HTML e Atributos de TAG

TAG em HTML são comandos que serão utilizados para interpretar alguma ação no documento HTML, como uma inserção de imagem,
vídeo áudio, parágrafo, título, etc.
Tudo que diz respeito a como o site será estruturado será referente as TAG do HTML, é ela que forma o "esquelo ou planta" de um website.
As TAG são declaradas da seguinte maneira:

<p>Aqui representa um parágrafo</p>

As TAG são declaradas dentro de um sinal de maior e menor "<>", e normalmente são fechadas com uma barra invertida "</>",
mas também há tag que não são fechadas, como o <br> e o <img>. O que diferencia uma tag com fechamento com uma que
não tem fechamento e o conteúdo, TAG que conterá conteúdo, normalmente tem fechamento, já as que não há conteúdo não conterá
um fechamento, como por exemplo o parágrafo, dentro da TAG será inserido uma texto, já a tag br e a img não conterão conteúdos,
porém terá outra função como quebrar linha ou buscar uma imagem, seja ele local ou através de uma URL.

As TAG poderão definir um elemento, como por exemplo imagem, toda imagem terá uma largura ou altura, que no documento ficará
parecendo um quadrado, esses tamanhos dos elemento poderão posteriormente ser alterados de acordo com o esperado resultado final
que o site terá. O que e importante saber de príncipio é que há TAG que quando definidas terá um comportamente diferente da outra
em relação ao espaço, poderá ter TAG que ocupará uma área horizontal completa e terão TAG que ocuparão apenas o espaço necessário para
inserir o elemento.
São os casos dos elementos block e inline, sem o block o que contél largura e altura e o inline o que usa somente o espaço necessário.

Atributos são informações que passamos na TAG para que ela se comporte da maneira esperada. Existem atributos globais (que funcionam em
todas as TAG) e específicos (que são direcionados para cada TAG, através de especificação).
Os Atributos possuem nome e um valor, que normalmente são usados frequentemente e também há outros mais dificeis de serem usados.
São declarados da seguinte maneira:

<img src="valor do atributo">

No caso src=" " seria o atributo e o valor seria colocado dentro das aspas, que normalmente se refere a um caminho ou quantidade. Há vários
tipos de atributos além do src=" ".

No HTML há uma estrutura básica que consiste em todo arquivo HTML, normalmente já vem até feita nos editores de texto mais modernos.

<!DOCTYPE html>
<html lang="pt-br">
     <head>
          <meta charset="utf-8"/>
          <title">Título da página</title>
     </head>
     <body>
          …..conteúdo da página
     </body>
</html>

Há também uma confusão na hora de codificar arquivos HTML, pois pode haver a necessidade de utilizar caracteres que são utilizados como códigos
para esses casos utiliza-se codificações para substituir os caracteres especiais, sendo alguns deles:

quot que é utilizado para representar aspas ";
apos que é utilizado para representar Apóstrofe '';
amp que é utilizado para representar "E" comercial;
lt que é utilizado para representar o sinal de menor que;
gt que é utilizado para representar o sinal de maior que;
nbsp que é utilizado para representar um pequeno espaço no texto.

Enfatizando que todos eles tem que serem antecedidos de um e comercial (&), senão não funcionam.

Todo código HTML pode ser válidado pelo W3C no link: https://validator.w3.org/

TAG e suas funções

<!DOCTYPE html>

Serve para indicar que o documento HTML estará utilizando a versão HTML 5.

<html>

É utilizada para iniciar um documento HTML, normalmente vem com o atributo lang, que servirá para indicar o idioma do documento.

<head>

Indicará o cabeçalho do do site, e se encontra dentro da TAG <html>.

<title>

Serve para colocar o título do site, normalmente utilizada já dentro do cabeçalho do site.

<meta>

Serve para definir informações do documento, como o autor, palavras-chave, jogo de caracteres, etc. Utilizada no cabeçalho.
Exemplo:

<meta charset="utf-8" /> preferido em HTML 5.
<meta charset="ISO-8859-1" /> preferido em HTML 4.

Especifica palavras chaves
<meta name="keywords" content="ia, objetos, classes" />

Descrição do conteúdo
<meta name="description" content="exemplo da aula 2" />

<body>

Serve para iniciar o corpo do site, é aonde estará todo o conteúdo de fato de um website.

<br>

Serve para quebrar uma linha.

<hr>

Serve para quebrar uma linha e colocar uma linha divisória.

<h1>, <h2>...

Serve para indicar um título no documento, e ele vai até 6, h1, h2, h3, h4, h5, h6 onde o primeiro define uma importância maior.

<div>

Serve para indicar um espaçamento genérico, que pode ser utilizado de diversas formas.

<!-- ... -->

Serve para inserir um comentário dentro do código HTML, que não será visível quando o navegador processar o documento.
